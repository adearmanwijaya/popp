## Installation

Install with python

1. Download python
2. Install Module (pip install requests colorama)
3. Buka bot popp planet
4. Inspect > Application > session storage > __telegram__initParams
5. Copy value tgwbappdata
6. Paste di query.txt
7. python popp.py
